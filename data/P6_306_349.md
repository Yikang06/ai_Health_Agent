# 第三部分 平衡膳食模式和膳食指南编写说明

## 一、膳食指南修订指导思想和概况

2020年，中国营养学会膳食指南修订专家委员会以营养科学原理为基础，针对当前主要的公共卫生问题，紧密结合我国居民膳食消费和营养状况的实际情况，在《中国居民膳食指南（2016）》的基础上进行修订，提出第5版《中国居民膳食指南（2022）》。

### （一）修订原则和程序

**修订原则：**
- 以大众营养健康需求为根本，以营养科学原理、食物和健康关系的最新科学证据为根据
- 以“平衡膳食模式”为核心指导思想，优先考虑我国目前突出的膳食营养共性问题
- 以食物为基础，实践指导内容充分重视实用性、可操作性及食物系统的可持续发展

**覆盖范围：** 以2岁以上健康人群为目标，补充特别关注人群（孕妇和乳母、婴幼儿、儿童、老年人）及素食人群的膳食指南。

### （二）主要修订内容

与2016版相比，《中国居民膳食指南(2022)》的主要修订特点如下：

**1. 新增健康饮食方式建议**
由原6条核心推荐修订为8条膳食准则，新增：
- 规律进餐
- 会选会烹
- 公筷分餐
- 杜绝浪费
- 饮食卫生

> 💡 **背景说明：** 我国不健康生活方式仍然普遍存在，居民超重和肥胖问题不断凸显，与外卖、在外就餐、食物知识缺乏和不正确的减肥方式流行有关。新冠肺炎疫情也提示要重视公共卫生和个人卫生，坚持公筷公勺、分餐或份餐等措施，避免食源性疾病发生和传播。

**2. 应对老龄化——新增高龄老年人膳食指南**
随着人均预期寿命不断增长，高龄（≥80岁）、衰弱老年人的比例逐渐增加。这一群体身体各系统功能显著衰退，营养不良发生率高，慢性病发病率高，需要更加专业、精细和个性化的膳食营养管理指导。

**3. 强调东方健康膳食模式**
我国江南及东南沿海一带（浙江、上海、江苏、福建等）膳食特点：
- 烹调清淡少盐
- 食物多样，谷物为主
- 丰富的蔬菜水果
- 经常吃鱼虾等水产品、大豆制品和奶类
- 该地区高血压等慢性病患病率低，预期寿命较高

《中国居民膳食指南(2022)》首次给予定义并推荐“东方健康膳食模式”。

**4. 更新定性定量食物选择和膳食营养新概念**
修订和增加了30余个定量和定性描述性用语，如：
- 食物概念：合理膳食、健康膳食、全谷物、素食、营养素密度等
- 含量描述：富含、含有、高脂、高能量、低盐等
- 摄入频度：多吃(喝)、少吃(喝)、适量、控和限吃(喝)

**5. 使用最新科学证据**
补充了2014年7月至2020年10月国内外有关食物与健康研究的新证据，结合《中国居民营养与慢性病状况报告(2020年)》数据。

**6. 修改完善图形和食谱的可视化**
进一步完善平衡膳食宝塔、平衡膳食餐盘等图形，拍摄定量食谱图案，使之更加可视化、现代化。

### （三）科学证据收集和引用

**1. 证据等级评价方法**

参照WHO指南制定手册(2014，第二版)，证据评价方法主要包括：提出问题、收集文献证据形成证据体、证据体（强度等级）评价和结论推荐。

研究设计水平分值由高到低：
系统综述（或Meta分析）→ 随机对照试验(RCT)研究 → 队列研究 → 有对照横断面研究 → 无对照横断面研究 → 个人经验（或专家意见、个例报告等）

**表3-1 食物与健康文献证据综合评价表**

| 推荐等级 | 结论可信度 | 科学价值 | 评价标准 |
| :---: | :--- | :--- | :--- |
| **A** | 由该证据体得出的结论是可信的 | 证据质量高，应用价值大 | 5项为优秀 |
| **B** | 在大多数情况下，该证据体的结论是可信的 | 证据质量较高，应用价值较大 | 3~5项为优秀或良好 |
| **C** | 该证据体的结论可能是可信的，但资料少，应用时应加以注意 | 部分证据质量较高，有一定应用价值 | 1~2项为优秀或良好 |
| **D** | 该证据体不能得出结论或结论不可信 | 证据不足或质量较差，无明显应用价值 | 5项评价指标中等，无1项评为优秀或良好 |

**证据强度等级具体描述：**

- **A. 确信的证据(convincing evidence)：** 证据多，样本量足够大，包括高质量RCT研究、前瞻性队列研究等。Meta分析显示一致性结果，很少或没有相反结果。膳食摄入量（暴露）和疾病之间密切相关，生物学上真实可信。

- **B. 很可能的证据(probable evidence)：** 流行病学证据清楚显示食物摄入或膳食因素（暴露）和疾病之间的关系，但尚存一些不同文献证据或部分研究设计有缺陷，不能得出完全肯定的结论。相关性在生物学上很可能是真实的。

- **C. 可能的证据(possible evidence)：** 以病例对照研究和横断面观察性研究为主，RCT、队列研究资料较少。结果有一定相关性，但需更多研究证实，生物学真实性是可能的。

- **D. 证据不足(insufficient evidence)：** 基于少数研究结果的提示，不足以建立食物摄入量（暴露水平）和疾病之间的相关性。需更多精心设计的研究证实。

> ⚠️ **局限性：** 研究证据收集1997—2020年主要文献，超出该时间范围未纳入。二次研究文献及部分原始研究不能获得详细食物摄入量，对描述量效关系有一定局限性。大量原始研究证据多来自队列研究，仅能得出某食物可能降低或增加某疾病发病风险，尚不能明确因果关系。详细内容可查阅《中国居民膳食指南科学研究报告(2021)》。

### （四）应用概念及定义

概念确定三原则：引用国家标准/国际组织标准 → 引用权威专业书籍定义 → 指南修订专家委员会统一规范。

**表3-3 常见膳食定性术语和概念描述**

| 名词 | 定义 |
| :--- | :--- |
| **膳食模式** | 亦称膳食结构，指膳食中各食物的品种、数量及其比例和消费的频率。受人口、农业生产、食物流通、食品加工、消费水平、饮食习惯、文化传统、科学知识等多种因素影响。 |
| **平衡膳食模式** | 根据居民膳食营养素参考摄入量、居民营养与健康状况所推荐的食物种类和比例，能最大限度地满足不同年龄阶段健康人群的生理和营养健康需要而设计的膳食。 |
| **合理膳食** | 在平衡膳食基础上，考虑到健康状况、地域资源和生活习惯、信仰等情况而调整的膳食，能较好地满足不同生理状况、不同人群的营养与健康需要。 |
| **健康膳食** | 针对健康结局（慢性疾病发生、预期寿命等）而言，特点常包括少油盐、少深加工食品、多蔬果全谷物等。平衡膳食、地中海饮食以及DASH饮食通常被认为是健康膳食。 |
| **东方健康膳食模式** | 以我国浙江、上海、江苏、福建等地区为代表，膳食特点以食物多样、清淡少油为主，尤其以丰富蔬菜水果、多鱼虾海产品、多奶类和豆类为主要特征。该地区慢性病发病率和死亡率较低，预期寿命较高。 |
| **素食** | 不包含动物性肉类食物的膳食模式。本指南涉及全素食（纯素食）和蛋奶素食者。 |
| **全谷物** | 未经精细加工或虽经碾磨/粉碎/压片等处理，仍保留了相对完整谷粒所具备的胚乳、胚芽、麸皮组成及天然营养成分的谷物。 |
| **全谷物食品** | 食品中全谷物重量不低于51%的食品，其全谷物原料为100%全谷物。 |
| **添加糖** | 在加工和制备食品时，添加到食物或者饮料中的糖或糖浆，包括蔗糖（白糖、砂糖、红糖）、葡萄糖、果糖（结晶或非结晶）、各种糖浆等。 |
| **纯能量食物** | 能量来源单一，除能量外几乎不含有其他营养素的食物，如精制糖、淀粉、动植物油脂、酒等。 |

**表3-4 常见食物定性和定量描述**

| 名词 | 定义 |
| :--- | :--- |
| **多吃(喝)** | 该食物是平衡膳食的基本组成部分。调查显示多数人群摄入不足且增加摄入对健康有益时，建议“多吃”。通常指每天必须吃或倡导比以前摄入量多。 |
| **常吃(适量)** | 该食物是平衡膳食的基本组成部分，过少或过高摄入可能增加慢性病风险。建议“适量”摄入，即膳食指南中的推荐量。“常吃”通常指周摄入频率为3~5次。 |
| **少吃(喝)** | 该食物是平衡膳食的组成部分，但过高摄入能增加慢性病发生风险，且调查显示大部分人群摄入量过高。日常需特别注意减少食用，如盐、油。 |
| **控和限吃(喝)** | 该食物是或不是平衡膳食的组成部分，过高摄入对健康产生不利影响。调查显示过量摄入普遍存在，需“控制”或“限制”食用，如糖、酒。 |
| **营养素密度** | 评价食品营养价值的一种指标。营养素密度 = 食物中某营养素含量 ÷ 该食物能量 × 1000（折合成每1000kcal能量的营养素质量单位数） |
| **能量密度** | 100g食物所含能量值(kJ/100g或kcal/100g)，与食物水分和脂肪含量密切相关。 |
| **高能量** | 通常指提供400kcal/100g以上能量的食物。 |
| **高盐** | 通常指钠含量≥800mg/100g的食品，尤指零食和膨化食品类。 |
| **高糖** | 指饮料糖含量≥11.5g/100ml，固体饮料按冲调后液体中糖含量计。 |
| **高油/高脂** | 通常指脂肪含量≥20g/100g的食品。 |
| **低盐** | 钠含量≤120mg/100g(固体)或100ml(液体)。 |
| **低糖** | 碳水化合物(糖)≤5g/100g(固体)或100ml(液体)。 |
| **瘦肉** | 脂肪含量≤10%的畜肉和禽肉类。 |

## 二、平衡膳食模式研究

### （一）我国健康膳食研究

我国传统膳食特点是以植物性食物为主，膳食纤维含量丰富，缺陷是谷类食物摄入量过多（以前高达80%以上），动物性食物摄入量偏少，奶类和水果长期缺乏。随着生活水平提高，膳食结构发生变化，主要表现为畜肉类和油脂消费过多，粗杂粮、薯类消费锐减，导致营养素摄入失衡和肥胖等慢性病高发。

**不合理膳食是造成我国心血管疾病死亡和疾病负担的重要危险因素之一：**
- 2017年我国约**260万**心血管疾病死亡归因于膳食因素，较2007年增长**38%**
- 心血管病负担超过**8500万**伤残调整生命年(DALYs)，其中约**5600万**归因于不合理膳食

> 📊 **全球对比（2019年《柳叶刀》）：** 全球近20%的死亡是因为吃的食物不健康导致。中国因饮食结构问题造成的心血管疾病死亡率、癌症死亡率均为世界人口前20位大国中的首位。造成死亡的不合理饮食习惯前三位：**高钠饮食、低全谷物饮食、低水果饮食**。

中国人群不同膳食模式对健康结局影响研究结果显示，**浙江、上海、江苏、广东、福建等南方膳食模式**特点的人群发生超重肥胖、2型糖尿病、代谢综合征和脑卒中等疾病的风险均较低，心血管疾病和慢性病死亡率较低，期望寿命较高。

### （二）中国居民平衡膳食和构成

平衡膳食模式是经过科学设计的理想膳食模式，是中国居民膳食指南的核心。

**设计依据：**
1. 符合营养科学原理和中国居民膳食营养素参考摄入量
2. 结合最新居民营养与健康研究数据
3. 参考食物与健康关系证据研究
4. 考虑我国食物资源、饮食文化特点和食物系统可持续发展

#### 1. 中国居民平衡膳食模式的特点

**（1）食物多样**
五大类人体必需基本食物：谷薯类、蔬菜水果类、禽畜鱼蛋奶类、大豆坚果类及烹调用油盐。推荐每周25种以上食物。

> 💡 传承“五谷为养、五果为助、五畜为益、五菜为充”的膳食搭配原则。

**（2）植物性食物为主**
- 谷薯类提供能量占总能量的**50%**左右
- 蔬菜、水果、大豆、坚果占总体膳食比例较高

**（3）动物性食物为辅**
保障优质蛋白质摄入，弥补植物性食物中脂溶性维生素、维生素B₁₂、锌、硒等微量营养素不足，预防因动物性食物过多引起的心脑血管疾病及某些癌症风险增加。

**（4）少油盐糖**
- 食盐：**<5g/天**（DRIs 2013建议钠预防慢性病不超过2000mg，相当于5g盐）
- 烹调油：**25~30g/天**
- 添加糖：**<25g/天**（最好）

#### 5. 食物来源和膳食模式分析

平衡膳食模式以植物性食物为主，是一种比较经济并有利于可持续发展的膳食模式。膳食指南鼓励：

#### 1. 中国居民平衡膳食宝塔（2022）

图3-6 中国居民平衡膳食宝塔(2022)（略）

宝塔共分5层，各层面积大小体现5大类食物和食物量的多少。

**第一层 谷薯类食物**
- 碳水化合物提供总能量的50%~65%
- 推荐摄入量（1600~2400kcal）：谷类200~300g/d（其中全谷物和杂豆类50~150g），薯类50~100g
- 谷类包括小麦、稻米、玉米、高粱等及其制品
- 全谷物保留了天然谷物的全部成分
- 杂豆包括红小豆、绿豆、芸豆等
- 薯类包括马铃薯、红薯等，可替代部分主食

**第二层 蔬菜水果**
- 推荐摄入量（1600~2400kcal）：蔬菜≥300g/d，水果200~350g/d
- 深色蔬菜（深绿、深黄、紫色、红色等）占总体蔬菜1/2以上
- 深色蔬菜一般富含维生素、植物化学物和膳食纤维
- 推荐吃新鲜水果，鲜果供应不足时可选择含糖量低的干果制品和纯果汁

**第三层 鱼、禽、肉、蛋等动物性食物**
- 推荐摄入量（1600~2400kcal）：共计120~200g/d
- 畜禽肉40~75g/d，少吃加工类肉制品，尽量选择瘦肉或禽肉
- 水产品40~75g/d，富含优质蛋白质、脂类、维生素和矿物质
- 蛋类：推荐每天1个鸡蛋（约50g），**吃鸡蛋不能丢弃蛋黄**（蛋黄含有胆碱、卵磷脂、胆固醇、维生素A、叶黄素、锌、B族维生素等）

**第四层 奶类、大豆和坚果**
- 推荐摄入量（1600~2400kcal）：奶类≥300g/d，大豆和坚果共25~35g/d
- 坚果每周70g左右（约每天10g）
- 全球奶制品消费中，我国居民摄入量一直很低

**第五层 烹调油和盐**
- 烹调油：25~30g/d，应多样化，经常更换种类
- 食盐：<5g/d
- 酒和添加糖不是膳食组成的基本食物，应尽量避免

**身体活动和饮水**
- 饮水：低身体活动水平成年人每天至少1500~1700ml（7~8杯）
- 高温或高身体活动水平条件下适当增加
- 一天中饮水和整体膳食水摄入共计2700~3000ml
- 身体活动：每天至少相当于快步走6000步以上，每周最好进行150分钟中等强度运动（如骑车、跑步、庭院或农田劳动）

## 三、平衡膳食模式的应用

### （一）膳食实践和指导原则

**1. 个人饮食和生活方式实践**
- 设计平衡膳食，自我管理一日三餐
- 了解并实践“多吃”的食物
- 了解并控制“少吃”的食物
- 合理运动和保持健康体重
- 评价个人膳食和生活方式，逐步达到理想要求

**2. 公共营养和大众健康指导**
- 营养教育实践资源和教材
- 发展营养相关政策和标准的基础
- 创造膳食计算和资源的新工具
- 科学研究、教学、膳食管理的指导性文件
- 全民营养周、社区健康指导、健康城市等健康促进科学资源
- 慢性病预防和健康管理的行动指南
- 《健康中国行动(2019-2030年)》“合理膳食行动”落实的保障

**3. 营养教育与健康促进关键点**
1. 食物多样、平衡膳食的原则
2. 提倡和鼓励“多吃”的食物
3. 提倡和建议“少吃”的食物
4. 饮食行为和文明：公筷分餐、节俭不浪费
5. 鼓励实践，培养良好饮食习惯
6. 特别提及：合理运动、能量平衡、估量食物、公筷分餐制、生态环境
